# [krapac.cz](https://krapac.cz)

Personal photography portfolio website, now migrated from Gatsby to Astro.

Based on the [Strata template](https://www.gatsbyjs.org/starters/codebushi/gatsby-starter-strata/).

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
